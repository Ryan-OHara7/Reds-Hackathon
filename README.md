# Reds-Hackathon
This repository contains the results of the 2024 Cincinnati Red's Hackathon. Our recommendation was implemented and placed into the bullpen where he found early success and still plays today in the 2026 season.

CINCINNATI REDS HACKATHON
o	Analyzed advanced pitching metrics and machine learning outputs to identify pitchers who would benefit from role changes
o	Recommended a bullpen transition for a specific pitcher based on data-driven insights; the team implemented the recommendation for the 2025 season, where he achieved early success


Used advanced pitching metrics sourced from fangraphs.com. Used a combination of filtering and clustering models to identify Ghram Ashcraft as a pitcher who could benefit from a role change from starting pitcher to set up/high leverage bullpen pitcher.
